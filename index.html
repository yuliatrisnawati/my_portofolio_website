Fashion Lens
The "Fashion Lens" project is a comprehensive mobile application designed to revolutionize the online fashion shopping experience. The app features a clean and modern interface, with the central welcome screen showcasing a friendly illustration of a shopper interacting with the app. The application allows users to explore various fashion items, view detailed product images, and seamlessly browse through different categories. With "Fashion Lens" users can enjoy a personalized and engaging shopping journey, making it easier than ever to find and purchase their favorite fashion pieces.
Daily Clime
"Daily Clime" is a web-based application that provides an image upload feature for daily weather needs. Users can upload photos from their devices using the camera or choose files from their storage. This application is designed to help users share local weather conditions through images, making the user experience more interactive and informative.
CampusEASE
"CampusEASE" is a mobile application designed to simplify campus life by providing essential features for students. On the main page, users are greeted with a personalized message and can search for various events using the search feature. The app offers two main categories: "Correspondence" and "Events," making it easy for users to find information related to campus letters and events. This application aims to enhance student engagement by providing easy access to event information and other campus services.
